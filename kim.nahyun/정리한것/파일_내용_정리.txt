============================================================
  SKAX 이탈 예측 프로젝트 — 파일 내용 정리
  기간: 2025년 4월 8일 ~ 4월 25일
============================================================

[프로젝트 개요]
  SKAX 스트리밍 서비스의 멤버십 데이터를 분석하여
  이탈(Churn) 유저를 예측하는 머신러닝 모델을 구축하는 프로젝트.

  사용 데이터:
  - Membership.csv    : 구독자 정보 (결제금액, 기기, 상품코드, 이탈방지 여부 등)
  - View_History.csv  : 유저별 시청 이력
  - Movie_Master.csv  : 영화 메타데이터 (장르 등)
  - User_Mapping.csv  : 유저 매핑 테이블
  - features.csv      : 분석에 사용된 최종 피처 테이블 (Membership + 시청행동 + 파생변수)
  - 통합분석_데이터베이스.db : 위 테이블을 통합한 SQLite DB (~130MB)

  이탈 정의:
  - repurchase == NaN  →  이탈 (is_churn = 1)
  - repurchase == 'O'  →  잔류 (is_churn = 0)
  - 전체 이탈률: 약 34%

[1단계] 01_초기EDA_스트리밍사용자행동분석.ipynb   (Apr 8)
------------------------------------------------------------
  목표: 멤버십 데이터 구조와 기본 분포 파악

  주요 내용:
  1. 데이터 컬럼 설명 (user_no, product_cd, amount, billing_method 등)
  2. 결측치 분석 — is_churn_prevented, repurchase 등 일부 컬럼 결측치 많음
                   (빈 값 = "해당 없음" 의미일 가능성 높음)
  3. 범주형 변수 분포 분석 (상품코드, 결제기기 등)
  4. 수치형 변수 분포 분석 (결제금액, 나이 등)

  6. 이탈방지(is_churn_prevented) 처리 여부에 따른 변수 비교
  7. 수치형 변수 상관관계 히트맵
  8. 상품별 비교 분석 (판매량, 평균금액, 이탈방지율)

  주요 인사이트:
  - 특정 상품에 가입 집중
  - PC / Android / iOS 중 주력 결제 채널 파악
  - 구독 기간 약 30일(월정액) 집중

[10단계] 10_최종모델링_이탈예측.ipynb   (Apr 24)
------------------------------------------------------------
  목표: 최종 피처 기반 4개 모델 비교 + 최적 모델 선정

  주요 내용:
  1. 데이터 준비: features.csv 로드, 불필요 컬럼 제거, 타깃 분리
  2. 인코딩: 범주형 LabelEncoder, 결측치 처리
  3. Train/Test Split — 8:2, stratify (이탈 비율 유지)
  4. 모델 학습:
     - Logistic Regression (선형 기준점)
     - Random Forest (트리 앙상블)
     - XGBoost (Gradient Boosting)
     - LightGBM (경량 Gradient Boosting)
  5. 모델 성능 비교: Accuracy / F1 / ROC-AUC
     (불균형 데이터이므로 ROC-AUC, F1 핵심 지표)
  6. 피처 중요도: LightGBM + Random Forest 비교
  7. 임계값 튜닝 (LightGBM)
     - 기본 0.5 → 최적 임계값 탐색
     - 목적에 따라: Recall 우선(놓치지 않기) vs Precision 우선(비용 절감)
  8. 단계별 피처 추가 AUC 비교:
     ┌──────────────────┬────────────────────────────────────┐
     │ 단계             │ 추가 피처                          │
     ├──────────────────┼────────────────────────────────────┤
     │ 생깡통           │ 멤버십 기본 12개 (리케이지 제외)   │
     │ 베이스라인       │ + 기본 시청 9개                    │
     │ 결측치 처리 후   │ + 완주율/유지율/최근성 8개         │
     │ 영화 데이터 포함 │ + 장르 선호도 14개                 │

[11단계] 11_비지도학습_군집분석.ipynb   (Apr 25)
------------------------------------------------------------
  목표: 비지도 학습 개념 정리 및 실습 (교재 학습)

  주요 내용:
  1. 비지도 학습의 종류와 도전과제
  2. 데이터 전처리 & 스케일 조정
     - StandardScaler: (x-평균)/분산, 이상값에 민감
     - RobustScaler: (x-중간값)/사분위, 이상값에 강건
     - MinMaxScaler: (x-min)/(max-min), 0~1 사이
     - Normalizer: 행 기준, 벡터 길이=1
     - QuantileTransformer, PowerTransformer
  3. 훈련/테스트 스케일 동일하게 맞추기
     (fit은 훈련 데이터에만! 테스트는 transform만)
  4. 차원 축소 & 특성 추출
     - PCA (주성분 분석): 분산 최대 방향으로 차원 압축
     - 고유얼굴(eigenface): 얼굴 이미지 특성 추출
     - NMF (비음수 행렬 분해): 오디오/텍스트 패턴 추출
     - t-SNE: 2차원 시각화용 매니폴드 학습
