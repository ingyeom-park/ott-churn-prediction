OTT 고객 이탈 예측 — 전처리 전후 비교
1. user_no 중복 제거
동일 user_no가 여러 행으로 존재하는 338건 처리. duration_days가 가장 큰 행을 유지하고 나머지 제거.
18,183건 → 17,845건
2. 타겟 인코딩 — repurchase
O(재결제) → 1, NaN(이탈) → 0 으로 변환.
재결제 11,767건(65.9%) / 이탈 6,078건(34.1%)
3. 통화 통일 — 달러 → 원화
iOS 결제(pk_1508·pk_1506·pk_1507)는 달러($)로 청구됨. product_cd 기반 tier 정가로 매핑.
베이직 $9.99 → 7,900원 / 스탠다드 $13.49 → 10,900원 / 프리미엄 $16.49 → 13,900원
4. concurrent_streams 이상값·결측 제거
상품 tier가 1(베이직)·2(스탠다드)·4(프리미엄)만 존재.
concurrent_streams=3 (입력 오류 7건) + NaN (59건) = 총 66건 제거.
5. is_user_verified 결측 처리
결측(약 601건)을 N(미인증)으로 대체 — 인증 정보 없는 경우 미인증으로 간주.
6. 이진 컬럼 인코딩
O(해당) → 1, NaN(미해당) → 0 / Y → 1, N → 0 으로 변환.
promotion_yn · is_churn_prevented · is_user_verified 3개 컬럼 적용.
7. age 이상값 및 기본값 처리
이상값(>100): 2건 → NaN
결측: 150건 → NaN
age=40 + 미인증 3,621건 중 초과분: 인증 유저 내 age=40 자연비율 10.37% 적용 → 376건 유지, 3,245건 → NaN
전체 NaN 3,397건 → 인증 유저 나이 비율대로 정확히 배분
8. gender 처리 — N·결측 → F/M 비율 배분
N(미확인 2,749건) + 결측(152건) = 2,901건을 F/M 비율(F 62.56% / M 37.44%)로 배분.
처리 후: F 11,164건(62.6%) / M 6,681건(37.4%)
9. duration_days=0 제거
당일 해지(duration_days=0) 399건 제거 — 실제 이용 이력이 없는 당일 취소 건으로 분석 대상에서 제외.

전처리 후 EDA — 인증 vs 미인증 비교

age 분포 — 인증 고객 vs 미인증 고객
전처리 후 미인증 고객의 age도 인증 유저 비율대로 배분되어 두 집단의 나이 분포가 유사해짐. age=40 spike 사라지고 20~35세 중심으로 정상화.
gender 분포 — 인증 고객 vs 미인증 고객
전처리 후 N이 제거되어 F/M만 남음. 인증·미인증 고객 모두 동일한 F/M 비율(약 63:37) 유지.
